# 🌍 AMPYtranslate

AMPYtranslate is a **speech-to-speech language translator** built using **Python** and **Streamlit**. It converts spoken input into translated speech in over **100 languages** using Google Speech Recognition, Google Translate, and Google Text-to-Speech.

## 🚀 Features
- 🎙️ Speech-to-Text
- 🌐 Multi-language Translation
- 🔊 Text-to-Speech Output
- 💻 Interactive Streamlit Interface

## 🛠️ Tech Stack
- Python
- Streamlit
- SpeechRecognition
- googletrans
- gTTS
- Pillow

## ▶️ Run

```bash
pip install -r requirements.txt
streamlit run app.py
```
